# Tech Blog

## Description
This application allow a user to sign up, login, and post their favorite tech articles. Users have the ability to comment on other's posts, as well as edit and delete their own posts.

## Technologies

* JavaScript
* Node.js
* Express.js
* MySQL2 
* Sequilize
* bcrypt 
* jest
* Handlebars.js
